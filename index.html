explain-tool apt 仓库

一键安装命令：

# 1. 添加 GPG 公钥
curl -fsSL https://Kriemseeley.github.io/explain_easily/explain-tool.gpg.key | sudo gpg --dearmor -o /etc/apt/keyrings/explain-tool.gpg

# 2. 添加软件源
echo "deb [signed-by=/etc/apt/keyrings/explain-tool.gpg arch=all] https://Kriemseeley.github.io/explain_easily stable main" | sudo tee /etc/apt/sources.list.d/explain-tool.list

# 3. 安装
sudo apt update && sudo apt install explain-tool
